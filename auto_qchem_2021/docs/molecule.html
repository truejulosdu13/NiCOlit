class molecule.molecule(input, isotopes_as_labels=False, max_num_conformers=1, input_type='string', input_format='smi', gen3D_option='best', min_fragment_dist=2)¶

Wrapper class for openbabel.OBMol class

__init__(input, isotopes_as_labels=False, max_num_conformers=1, input_type='string', input_format='smi', gen3D_option='best', min_fragment_dist=2)¶

Initialize the OBMol molecule, generate initial 3D geometry and do conformer search.

Parameters

  • input – string or file path

  • isotopes_as_labels – a flag to treat isotopes in the smiles string as atomic labels

  • max_num_conformers – maximum number of conformers to generate, use 1 (default) for no conformer search, a reasonable number of conformations to use is 30. Conformations are generated with genetic algorithm using pybel.ob.OBConformerSearch class.

  • input_type – “string” or “file”, in line with the input

  • input_format – any format supported by OpenBabel, e.g. ‘smi’, ‘cdx’, ‘pdb’, etc.

  • gen3D_option – “best”, “medium”, “fast” or “gen2D” (no 3D generation)

  • min_fragment_dist – minimum distance between molecular fragments for salts, no more than 2 molecular fragments are supported
