db_functions.descriptors(tags, presets, conf_option, substructure='') → dict¶

Retrieve DFT descriptors from the database

Parameters

  • tag (str) – metadata.tag of the db records

  • presets (list) – list of descriptor presets from ‘global’ (molecule level descriptors), ‘min_max’ (min and max for each atomic descriptor across the molecule), ‘substructure’ (atomic descriptors for each atom in the substructure)

  • conf_option (str) – conformer averaging option: ‘boltzmann’ (Boltzmann average), ‘max’ (conformer with highest weight), ‘mean’ (arithmetic average), ‘min’ (conformer with smallest weight), ‘any’ (any single conformer), ‘std’ (std dev. over conformers)

  • substructure (str) – substructure SMARTS string

Returns
